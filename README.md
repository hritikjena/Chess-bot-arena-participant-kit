# Chess-bot-arena-participant-kit
♟️ Chess Bot Arena – Participant Kit

Welcome to Chess Bot Arena (Sankalan 2026) 🚀
Your goal is to build a chess-playing bot using Python and compete against others!

⸻

🧠 What You Need to Do

You need to create a bot that:

* Takes the current board position (FEN)
* Decides the best move
* Returns the move in correct format

* 📂 Files Provided

* engine.py → Runs the match between bots
* bot_random.py → Sample basic bot
* my_bot.py → You will create/edit this file

⚙️ Setup Instructions

1. Make sure Python is installed (Python 3.8+ recommended)
2. Install required library:

pip install chess

3. Keep all files in the same folder

▶️ How to Run
python engine.py

⸻

🧠 How to Improve Your Bot

You can:

* Capture high-value pieces
* Avoid losing your pieces
* Look ahead 1–2 moves
* Use evaluation functions

⚠️ Rules

* ❌ No pre-built chess engines (Stockfish, etc.)
* ❌ No machine learning models
* ❌ No external APIs
* ❌ No invalid moves

⏱️ Time Limit

Your bot must return a move quickly (within a few seconds).
Slow bots may be disqualified.

🚀 Final Note

You are not training a model —
you are coding how your bot thinks.

Build. Battle. Dominate. ♟️🔥

⸻

If you get stuck, reach out in the WhatsApp group.

Good luck! 🚀
